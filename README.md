# pens'
**필기와 온라인 회의 기능을 통합한 브레인스토밍 플랫폼**

## 특장점
- 필기 공유를 통해 여러 사람이 함께 한 문서에서 작업할 수 있다.
- 실시간 음성채팅을 통해 자유롭게 소통할 수 있다.
- 그룹 기능을 통해 각 그룹의 목적에 맞게 문서와 음성 채팅방을 형성하여 사용 할 수 있다.
